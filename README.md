Проект встраиваемой системы цифровых часов на базе микроконтроллера Arduino. Система разработана с фокусом на точность отсчета времени, энергоэффективность и надежное сохранение пользовательских настроек.

## Описание
Проект представляет собой автономное устройство для отображения времени с использованием OLED-дисплея. В отличие от простых решений, здесь реализована логика работы с UNIX-временем, что упрощает математическую обработку временных интервалов и синхронизацию.

## Технологический стек
* **Язык:** C++ (Arduino Framework)
* **Периферия:** OLED-дисплей (протокол I2C)
* **Хранение данных:** EEPROM (энергонезависимая память)
* **Средства разработки:** Arduino IDE / PlatformIO

## Ключевые особенности
* **Математика времени:** Прямая работа с UNIX Timestamp для корректного вычисления времени и дат.
* **Оптимизация:** Использование аппаратных прерываний таймера для минимизации «дрейфа» часов и разгрузки основного цикла (`loop`).
* **Устойчивость:** Сохранение пользовательских параметров (настройки отображения, часовой пояс) в EEPROM; настройки не сбрасываются при отключении питания.
* **Интерфейс:** Работа с I2C-дисплеем через оптимизированные библиотеки для быстрой отрисовки графики.

## Инженерные вызовы (Challenges)
- **Точность:** Борьба с накоплением погрешности при длительной работе системы (реализована через прерывания).
- **Память:** Оптимизация записи в EEPROM для предотвращения износа ячеек памяти при частых изменениях настроек.
- **Интеграция:** Настройка стабильного соединения по шине I2C в условиях потенциальных помех.
