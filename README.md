# Unix Time Processor for Arduino

Программный модуль для обработки и отображения UNIX-времени на микроконтроллерах Arduino. Система позволяет вести точный отсчет времени, управлять событиями и сохранять состояние в энергонезависимую память.

## Функционал
* **Unix-to-Date Conversion:** Алгоритм преобразования UNIX timestamp в человекочитаемые дату (день/месяц/год) и время.
* **Precision Timing:** Использование прерываний таймера (`TimerOne`) для высокоточного счета времени без задержек.
* **Persistence:** Автоматическое сохранение текущего времени в EEPROM для корректного возобновления работы после перезагрузки.
* **CLI Control:** Управление через Serial-порт:
    * `G` — вывод текущей даты и времени.
    * `S` — установка новой даты (UNIX timestamp).
    * `O` — сброс буфера EEPROM.

## Технические детали
* **Прерывания:** Обработка `oneSecond()` через `TimerOne`.
* **Математика:** Расчет високосных годов реализован в цикле обработки дней месяца.
* **Память:** Минимизация количества циклов перезаписи EEPROM путем записи только при необходимости (реализовано через таймер `millis()`).

## Сценарии использования
Модуль можно использовать как основу для:
- Логгеров данных (Data Loggers).
- Таймеров полива или освещения.
- Часов реального времени (RTC) без использования аппаратного модуля RTC (при наличии стабильного питания).
